# 🐱 Благотворительный фонд поддержки котиков QRKot

Проект представляет собой REST API для управления благотворительными проектами и пожертвованиями, созданный с использованием **FastAPI**. Сервис позволяет создавать, просматривать, обновлять и удалять проекты по сбору средств для котиков, а также принимать пожертвования.

## 📌 Основные возможности

- **Управление проектами**:
  - ✅ Создание новых проектов
  - 📋 Получение списка всех проектов
  - ✏️ Обновление существующих проектов
  - 🗑️ Удаление проектов (с ограничениями)

- **Управление пожертвованиями**:
  - 💰 Создание пожертвований
  - 📊 Просмотр всех пожертвований

- **Безопасность и валидация**:
  - 🔒 Проверка уникальности имени проекта
  - ⚠️ Защита от редактирования закрытых проектов
  - 💸 Контроль сумм (нельзя установить цель ниже уже собранной)
  - 🛡️ Валидация входных данных



## 🚀 Быстрый старт

### Требования
- Python 3.9+
- FastAPI
- Uvicorn (для запуска сервера)

### Установка и запуск

1. **Клонируйте репозиторий**:
```bash
git clone https://github.com/skypixie/cat-charity.git
cd cat-charity
```
2. **Создайте и активируйте виртуальное окружение**
```bash
python -m venv venv
. venv/bin/activate
```

3. **Установите зависимости**:
```bash
pip install -r requirements.txt
```

4. **Примените миграции Alembic**:
```bash
alembic upgrade head
```

5. **Запустите сервер**:
```bash
uvicorn app.main:app
```

6. **Откройте документацию**:
   - Swagger UI: http://localhost:8000/docs
   - ReDoc: http://localhost:8000/redoc



## 📚 Документация API

### Проекты

| Метод | Эндпоинт | Описание | Статусы ответа |
|-------|----------|----------|----------------|
| `GET` | `/charity_project/` | 📋 Получить все проекты | 200 |
| `POST` | `/charity_project/` | ✅ Создать новый проект | 200, 400, 422 |
| `PATCH` | `/charity_project/{project_id}` | ✏️ Обновить проект | 200, 400, 404, 422 |
| `DELETE` | `/charity_project/{project_id}` | 🗑️ Удалить проект | 200, 400, 404, 422 |

**🔍 Особенности PATCH**:
- ❌ Нельзя редактировать закрытые проекты
- 💸 Нельзя установить `full_amount` меньше уже инвестированной суммы
- 🔤 Имя проекта должно быть уникальным

**⚠️ Особенности DELETE**:
- Нельзя удалить проект, в который уже внесены средства
- Нельзя удалить закрытый проект

### Пожертвования

| Метод | Эндпоинт | Описание | Статусы ответа |
|-------|----------|----------|----------------|
| `GET` | `/donation/` | 📊 Получить все пожертвования | 200 |
| `POST` | `/donation/` | 💰 Создать пожертвование | 200, 422 |
